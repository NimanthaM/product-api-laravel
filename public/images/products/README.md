# Product Images

This folder contains product images for the Aahaas Ecommerce application.

## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 500x500px (1:1 ratio)
- **File naming**: Use descriptive names (e.g., `headphones.jpg`, `smartwatch.png`)
- **File size**: Keep under 500KB for optimal loading

8. Laptop Stand - `laptop-stand.jpg`

## Adding New Images

1. Place your image in this folder (`public/images/products/`)
2. Update the database seeder with the new image path
3. Images will be accessible at: `http://localhost:8000/images/products/your-image.jpg`
